# Sobre el proyecto

## Objetivos

Desarrollar un e-commerce funcional con React que permita:
- Explorar productos por categoría.
- Gestionar un carrito de compras.
- Simular un flujo completo de compra.

Aplicando buenas prácticas de:
- Manejo de estado global.
- Componentización.
- UX/UI.
- Escalabilidad.

## Requerimientos

- Funcionalidad y UX: Experiencia de usuario fluida y profesional. 
- Uso de React y Hooks: Uso correcto de componentes funcionales, hooks básicos y personalizados y Zustand para el estado global. 
- Integración de API: Manejo efectivo del consumo de datos externos, incluyendo manejo de errores y estado de carga. 
- Estilos y diseño responsive: Diseño visual bien logrado, usando CSS Modules, y adaptable a diferentes dispositivos. 
- Control de versiones y despliegue: Historial de cambios en GitHub y despliegue en Vercel accesible públicamente.

## Tecnologías y librerías utilizadas

- HTML.
- CSS.
- Javascript.
- React.
- Vite.
- React Hook Form.
- Zustand.
- React Router.

### Uso de IA

Se utilizó inteligencia artificial generativa para:
- Mejorar la estructura del ecommerce.
- Resolver dudas y corregir errores.
- Generar estilos CSS.

## Roadmap

### Base del proyecto
- Configuración inicial con React + Vite.
- Estructura de carpetas escalable.
- Routing con React Router.
- Estilos con CSS Modules.

### Catálogo de productos
- Consumo de API con useFetch.
- Listado de productos por categoría.
- Página de detalle de producto.
- Buscador con filtro con useFilter.
- Navegación entre categorías.
- Selección de cantidades en la home, la página de categoría y la ficha de producto.

### Carrito de compras
- Implementación global con Zustand con useCartStore.
- Agregado de productos al carrito.
- Manejo de cantidades por producto.
- Eliminación de productos del carrito.
- Vaciado del carrito.
- Cálculo del total de items y monto.

### Checkout
- Formulario con React Hook Form.
- Limpieza del carrito al finalizar.
- Redirección a página de agradecimiento.

### Navegación y UX
- Navbar con categorías dinámicas.
- Indicador de cantidad en carrito (badge).
- Manejo de estados.
